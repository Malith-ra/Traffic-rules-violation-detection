Expressway Rules Violation Detection System Using Deep Learning

We have done research on detecting rules violation on the expressways in Sri Lanka. In this project a system is developed for this purpose using image and video processing techniques that can be used on the expressways of Sri Lanka. Our system contains following features.

Vehicle Type and License Plate Detection

This component can successfully identify the type of the vehicle like car, bus, van that enters to the expressway. And along with that the license plate numbers of those vehicles are also identified.

Parking Violation Detection

Speed violation detection means detecting the vehicles that are Parking without special permission or reasons of the express way.

No automated system has been introduced to detect illegal parking on the Sri Lanka Expressway.

Parking violation detection using CCTV video processing is an efficient method.

Lane Violation Detection

Lane violation detection means detecting the vehicles that are driving in the second lane(overtaking lane) when a vehicle is not passing another vehicle of the express way.

No automated system has been introduced to detect lane violation on the Sri Lanka Expressway.

Lane violation detection using CCTV video processing is an efficient method.

Speed Violation Detection

This component can successfully detect the speeds of the vehicles that are on the expressway and along with that this component can successfully recognize the vehicle colors of those vehicles. Then the vehicles that are exceeding the maximum speed limit of the expressway which is 100km/h are identified.

Reverse Violation Detection

This component can successfully identify the vehicles that are illegally driving in the reverse direction of the expressway.

An Interface For System

Our System can real time detect traffic rules violated vehicles. System performs high accuracy.
